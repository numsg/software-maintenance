# 数据库备份

这个工具主要是用来对数据做一个简单备份工作。可以备份数据字典还可以备份数据。

## 使用
这个工具使用十分简单，只需要配置一个简单的数据库连接字符串，选择需要备份的对象，导出结果可以得到一个chm文档。

 ![](image/db-back-1.png)

左边选择导出表结构，右边选择导出表数据。
